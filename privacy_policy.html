Política de Privacidade

WhatsApp Pro Exporter — Extensão para o Chrome

Última atualização: junho de 2025

1. Dados Coletados

Esta extensão não coleta, armazena, transmite ou compartilha nenhum dado pessoal do usuário com servidores externos.

As conversas exportadas são processadas inteiramente no seu navegador (localmente) e salvas diretamente no seu dispositivo como arquivo PDF. Nenhuma informação sai do seu computador.

2. Permissões Utilizadas

  • storage: Usado exclusivamente para manter o estado de autenticação da senha local da extensão, sem envio de dados externos.
  • activeTab: Permite à extensão interagir com a aba ativa do WhatsApp Web apenas quando o usuário aciona a exportação.
  • tabs: Necessário para capturar a tela visível da aba durante o processo de exportação para PDF.

3. Dados de Terceiros

Esta extensão não acessa, lê ou processa dados de conversas de forma automática. A exportação somente ocorre mediante ação explícita do usuário.

4. Segurança

A extensão conta com uma camada de proteção por senha local para evitar acesso não autorizado às funcionalidades.
